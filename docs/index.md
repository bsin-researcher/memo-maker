## What this does
1. **Generate a Snapshot** — price, 3M/12M momentum, 52-week drawdown.  
2. **Run an event study** — **T-1..T+3** vs **SPY** with abnormal returns.  
3. **Export & publish** — CSV download + copy-to-memo template for reproducible analysis.
